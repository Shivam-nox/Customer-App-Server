1. Admin Dashboard Endpoint
Admin Dashboard URL:

Your current Replit app URL (appears to be running on port 5000)
For production, it would be your deployed Replit URL
API Endpoint for Customer Registration:

POST /api/external/customer-registration
Full URL Example:

https://your-replit-app-name.replit.app/api/external/customer-registration
(Replace with your actual Replit URL)

üîê 2. Authentication
Method: API Key Authentication

Headers Required:

X-API-Key: zapygo-admin-2025-secure-key
Alternative format:

Authorization: Bearer zapygo-admin-2025-secure-key
Security Note: You can change this API key by setting the ADMIN_API_KEY environment variable in your admin dashboard.

üìã 3. Customer Data to Send
Required Fields:

name (string, 2-100 characters)
phone (string, 10-15 characters)
Optional Fields:

username (string, 3-50 characters)
email (valid email format)
businessName (string, max 200 characters)
industryType (string, max 100 characters)
Auto-Generated Defaults:

Email: {phone}@customer.zapygo.com (if not provided)
Business Name: {name}'s Business (if not provided)
Business Address: "Address to be updated"
Industry Type: "General Business" (if not provided)
KYC Status: "pending"
Role: "customer"
Credit Limit: "0"

Error Responses:

// Authentication Error (401)
{
  "error": "Unauthorized",
  "message": "Valid API key required. Set X-API-Key header or Authorization: Bearer <key>"
}
// Validation Error (400)
{
  "success": false,
  "error": "Validation failed",
  "details": [
    {
      "field": "phone",
      "message": "String must contain at least 10 character(s)"
    }
  ]
}
// Duplicate Customer (409)
{
  "success": false,
  "error": "Customer already exists",
  "message": "A customer with this phone number or email already exists"
}

JavaScript/Node.js Example:

const response = await fetch('https://your-admin-dashboard.replit.app/api/external/customer-registration', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'zapygo-admin-2025-secure-key'
  },
  body: JSON.stringify({
    name: 'John Doe',
    phone: '+91 9876543210',
    username: 'johndoe123'
  })
});
const data = await response.json();
